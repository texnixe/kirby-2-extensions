Kirbytext columns v2 (Beta)
===========================

## Requirements

- Kirby 2
- Bootstrap 3

### Back end usage:

```
(threecol: start)

Lorem ipsum dolor sit amet.

(threecol: break)

Lorem ipsum dolor sit amet.

(threecol: break)

Lorem ipsum dolor sit amet.

(threecol: end)
```

### Front end output

```
<div class="row">
  <div class="col-md-4">
    Lorem ipsum dolor sit amet.
  </div>
  <div class="col-md-4">
    Lorem ipsum dolor sit amet.
  </div>
  <div class="col-md-4">
    Lorem ipsum dolor sit amet.
  </div>
</div>
```
